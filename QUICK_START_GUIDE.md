### First Time Setup
If you're upgrading from the previous version:
1. **Automatic Migration**: Your existing learning path will be automatically converted to the new multi-path format
2. **Data Safety**: A backup is created before migration - no data loss!
3. **Enhanced Features**: You now have access to multiple path management

### Creating Your First Multiple Paths

#### 1. Create a New Learning Path
1. Click the **"Manage Paths"** button in the header
2. Select **"Create New Path"**
3. Fill in the details:
   - **Name**: Give your path a descriptive name (e.g., "Web Development Fundamentals")
   - **Description**: What will you learn in this path?
   - **Color**: Choose a color to visually identify your path
   - **Tags**: Add labels like "career", "beginner", or "certification"

#### 2. Switch Between Paths
**Desktop**: Use the dropdown in the header showing your current path
**Mobile**: Tap the colored circle next to your path name

#### 3. Organize Your Learning
- **Drag modules** from the left panel to your current path
- **Reorder modules** by dragging within your path
- **Track progress** independently for each path
- **Take quizzes** and earn certificates per path

## üÜò Troubleshooting

### Common Questions

**Q: What happened to my old learning path?**
A: It's been automatically migrated! Look for "Migrated Learning Path" in your path manager.

**Q: Can I have multiple paths with the same modules?**
A: Yes! Each path is completely independent. You can reuse modules across different paths.

**Q: Will my progress be lost if I delete a path?**
A: Yes, deleting a path removes all associated data. Consider archiving instead of deleting if you might want it later.

**Q: How do I backup my data?**
A: Use the export feature for individual paths, or your browser's localStorage backup tools for complete data backup.

**Q: Can I use this on multiple devices?**
A: Currently, data is stored locally in your browser. You can export/import paths to sync between devices manually.

### Getting Help
- **Migration Issues**: Check browser console for error messages
- **Missing Data**: Look in path manager for archived paths
- **Progress Problems**: Refresh the page to recalculate progress
- **Import Failures**: Validate JSON format before importing
